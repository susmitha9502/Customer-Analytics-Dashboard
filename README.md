# Customer Analytics Dashboard

## Project Overview

Developed an interactive Customer Analytics Dashboard using Python, Pandas, and Power BI to analyze customer purchasing behavior, sales trends, payment methods, and shopping mall performance.

This project transforms raw retail transaction data into meaningful business insights through data cleaning, exploratory data analysis, KPI generation, and interactive dashboard visualization.

---

## Tools & Technologies

* Python
* Pandas
* NumPy
* Power BI
* DAX
* Excel

---

## Dataset Information

The dataset contains customer retail transaction details including:

* Invoice Number
* Customer ID
* Gender
* Age
* Product Category
* Quantity
* Price
* Payment Method
* Invoice Date
* Shopping Mall

---

## Project Workflow

### 1. Data Cleaning & Transformation

Performed preprocessing using Python and Pandas:

* Removed duplicate records
* Checked missing values
* Converted data types
* Created calculated columns
* Generated Total Sales column

### 2. Exploratory Data Analysis (EDA)

Analyzed:

* Customer purchasing behavior
* Monthly sales trends
* Category-wise sales performance
* Shopping mall performance
* Payment method analysis

### 3. Dashboard Development

Created an interactive Power BI dashboard with:

* KPI Cards
* Pie Charts
* Donut Charts
* Line Charts
* Bar Charts
* Slicers & Filters

---

## Dashboard KPIs

* Total Sales
* Total Customers
* Total Orders
* Average Order Value

---

## Key Insights

* Clothing category generated the highest sales.
* Female customers contributed more revenue.
* Credit card payments were widely used.
* Top shopping malls generated significant revenue.
* Monthly sales trends helped identify customer purchasing patterns.
